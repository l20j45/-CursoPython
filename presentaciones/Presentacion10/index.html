Ya con los getter an setter podemos hacer uso de otra propiedad llamada sobrecarga de operadores

En la sobrecarga de operadores, lo que consiste en que en ocasiones se pueden tener el mismo nombre del metodo, y utilizar en diferentes formas, por lo general se utiliza con los constructores para poder tener varios constructores
recordemos que el constructor es quien nos ayuda a definir los parametros iniciales de los objetos, una forma en que podriamos hacerlo es la siguiente 

import time

class Fecha:
    def __init__(self, agnio,mes,dia):
        self.agnio = agnio
        self.mes = mes
        self.dia = dia
    
    @classmethod
    def hoy(cls):
        t = time.localtime()

        return cls(t.tm_year, t.tm_mon, t.tm_mday)

    def __str__(self):
        return '{}/{}/{}'.format(self.dia, self.mes, self.agnio)

    fecha1 = Fecha(2019,2,21)
    fecha2 = Fecha.hoy()
    print (fecha1)
    print (fecha2)
