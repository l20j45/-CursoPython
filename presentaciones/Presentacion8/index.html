Como acceder a los elementos?
Getter and Setter en python
por buenas praticas y para hacer uso de una propuedad llamada ENCAPSULAMIENTO
cada variable o atributo que tiene la clase tiene que tener un metodo para poder definir que se le va a poner informacion
de esta forma garantizamos siempre trabajar con esa informacion, y no alterar nuestro flujo del programa
Estos metodos son llamados, Getter (obtenci√≥n)
y Setter (establecimiento)
